# 🎵 YouTube Music App

A modern **YouTube Music streaming web application** built with **Next.js**.  
This app allows users to search and play music directly from YouTube using the **YouTube Data API v3**.

The goal of this project is to provide a simple and responsive interface for streaming music audio and easy download

---

## 🚀 Features

- 🔍 Search songs using YouTube
- ▶️ Play music directly from YouTube
- ⏸️ Play / Pause functionality
- 📃 Playlist UI
- ❤️ Favorite songs
- 📱 Fully responsive design
- ⚡ Fast performance using Next.js

---

## 🛠 Tech Stack

- **Next.js**
- **React**
- **TypeScript / JavaScript**
- **Tailwind CSS**
- **YouTube Data API v3**
